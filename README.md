# React + TypeScript + Vite

This is a template for a React application using TypeScript and Vite. It includes a basic setup for routing, state management with Redux, and a component library with Tailwind CSS.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You will need to have the following software installed on your machine:

*   [Node.js](https://nodejs.org/) (v18.x or later)
*   [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```

2.  Navigate to the project directory:

    ```bash
    cd your-repo-name
    ```

3.  Install the dependencies:

    ```bash
    npm install
    ```

    or

    ```bash
    yarn install
    ```

## Development

To start the development server, run the following command:

```bash
npm run dev
```

This will start the Vite development server with Hot Module Replacement (HMR) enabled. You can view the application by opening [http://localhost:5173](http://localhost:5173) in your browser.

### Available Scripts

*   `npm run dev`: Starts the development server.
*   `npm run build`: Builds the application for production.
*   `npm run lint`: Lints the codebase for potential errors.
*   `npm run preview`: Starts a local server to preview the production build.
*   `npm run test`: Runs the test suite.

## Building for Production

To build the application for production, run the following command:

```bash
npm run build
```

This will create a `dist` directory with the optimized production build of your application.

## Testing

This project uses [Vitest](https://vitest.dev/) for testing. To run the test suite, run the following command:

```bash
npm run test
```
